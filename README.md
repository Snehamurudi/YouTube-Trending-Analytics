# 📊 YouTube Trending Video Analytics
## 📌 Overview
This project analyzes YouTube trending video data to identify patterns in video performance, audience engagement, and content categories.

## 🛠️ Tools Used
* Python (Pandas, Matplotlib, Seaborn)
* SQL (SQLite)
* Tableau

## 📂 Dataset
YouTube Trending Dataset (India)

## 🔍 Key Analysis
* Data cleaning and preprocessing
* Sentiment analysis on video titles
* SQL-based category analysis
* Time-series trend analysis

## 📈 Dashboard
The Tableau dashboard shows:
* Category-wise performance
* Sentiment distribution
* Trending patterns over time

## 💡 Key Insights
* Certain categories consistently receive higher views
* Positive sentiment titles are more common
* Strong correlation between likes and views

## 🚀 Conclusion
This project provides insights into content strategies that can help creators maximize engagement.
